Item Management API

Simple Spring Boot REST API for managing items

Overview

About the API

This is a RESTful API built using Spring Boot that allows clients to perform basic CRUD operations on items. It is designed to be lightweight, scalable, and easy to integrate.

Base URL

Live Deployment

https://item-management-api-ikho.onrender.com

Endpoints

Available APIs

GET /api/items/{id}

{
  "id": 1,
  "name": "Laptop",
  "price": 50000
}

GET /api/items

[
  {
    "id": 1,
    "name": "Laptop",
    "price": 50000
  }
]

POST /api/items

{
  "name": "Phone",
  "price": 20000
}

PUT /api/items/{id}

{
  "name": "Updated Item",
  "price": 25000
}

DELETE /api/items/{id}

Usage

Sample cURL Request

curl -X POST https://item-management-api-ikho.onrender.com/api/items \
-H "Content-Type: application/json" \
-d '{"name":"Phone","price":20000}'
